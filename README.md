🌐 Humanize360 — Gestão Inteligente de Pessoas

O Humanize360 é uma plataforma completa de gestão de recursos humanos, desenvolvida para otimizar processos, melhorar a experiência dos colaboradores e apoiar decisões estratégicas com base em dados.

🚀 Sobre o Projeto

O Humanize360 foi criado com o objetivo de transformar a forma como empresas gerenciam pessoas, oferecendo ferramentas modernas para:

Gestão de colaboradores
Controle de ponto
Recrutamento e seleção
Avaliação de desempenho
Relatórios inteligentes

Tudo isso em um sistema intuitivo, responsivo e eficiente.

🛠️ Tecnologias Utilizadas
Frontend: HTML, CSS, JavaScript
Backend: Python (Flask)
Banco de Dados: SQLITE
